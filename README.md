UNICORN
====
可能性を追求した(する)フレームワーク  
主にスマホアプリケーション開発において「Fast Start」である事に重点を置いています。

##■利用開始方法
```
git clone git@github.com:saimushi/UNICORN.git
php UNICORN/src/lib/FrameworkPackage/core/UNICORN

```
